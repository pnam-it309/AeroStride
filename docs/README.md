# Tài liệu AeroStride

Thư mục `docs/` chứa các tài liệu kỹ thuật chi tiết của dự án. Bộ tài liệu tổng quan ở thư mục root được bổ sung theo cấu trúc tham chiếu để giúp onboarding, lần vết module và debug nhanh hơn.

## Bắt đầu từ đâu

- `../DOCUMENTATION_INDEX.md`: Mục lục tài liệu, nên đọc đầu tiên.
- `../PROJECT_OVERVIEW.md`: Bản đồ tổng quan dự án, module, luồng xử lý.
- `../DEVELOPMENT_WORKFLOW.md`: Cách bắt đầu một task, checklist trước khi commit.
- `../SECURITY_CHECKLIST.md`: Checklist bảo mật trước khi merge.
- `../TROUBLESHOOTING.md`: Cách xử lý các lỗi local phổ biến.
- `../MODIFICATION_LOG.md`: Mẫu ghi nhận thay đổi lớn.

## Tài liệu kỹ thuật trong thư mục này

- `backend_doc.md`: Cấu trúc backend Spring Boot theo layered architecture.
- `frontend_doc.md`: Cấu trúc frontend Vue/Vite và các vùng mã nguồn chính.

## Gợi ý đọc nhanh

1. Đọc `../PROJECT_OVERVIEW.md` để nắm tổng quan module.
2. Đọc `backend_doc.md` và `frontend_doc.md` để xác định file cần sửa.
3. Khi bắt đầu task mới, mở `../DEVELOPMENT_WORKFLOW.md`.
4. Trước khi merge thay đổi quan trọng, đi qua `../SECURITY_CHECKLIST.md`.
